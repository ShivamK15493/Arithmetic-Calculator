<h1>PROJECT DESCRIPTION</h1>
<p>A simple web-based Arithmetic Calculator built using HTML, CSS, and JavaScript.
This calculator performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.</p>

<h2>📌 Project Overview</h2>
<p>The Arithmetic Calculator is a beginner-friendly web application that allows
  users to enter two numbers and perform mathematical operations. The result is
  displayed instantly when the user clicks the desired operation button.</p>

<h2>🚀 Features</h2>
<li>Perform Addition</li>
<li>Perform Subtraction</li>
<li>Perform Multiplication</li>
<li>Perform Division</li>
<li>Display results instantly</li>
<li>Reset button to reload the calculator</li>

<h2>🛠️ Technologies Used</h2>
<li>HTML - Structure</li>
<li>CSS - Styling and layout</li>
<li>JavaScript - Logic and event handling</li>

<h2>📷 User Interface Components</h2>
<li>Two input fields for numbers</li>
<li>Four operation buttons</li>
<li>A result display area</li>
<li>A reset button</li>

<h2>🎯 Learning Objectives</h2>
<li>Understand the basic structure of a web page using HTML</li>
<li>Apply CSS styling to design a simple user interface</li>
<li>Use JavaScript to access and manipulate DOM elements</li>
<li>Perform arithmetic operations using JavaScript</li>
<li>Display results dynamically on the webpage</li>

<h2>✅ Conclusion</h2>
<p>The project demonstrates how HTML, CSS, and JavaScript
  can work together to create a simple interactive web application. Through this
  project, concepts of web development such as page structuring, styling, 
  DOM manipulation, and event handling are applied to perform arithmetic operations.
  It serves as a practical example for beginners to understand how user inputs can
  be processed and results displayed dynamically in a browser environment.</p>
